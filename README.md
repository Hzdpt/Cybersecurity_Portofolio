# Cybersecurity_Portfolio
Cybersecurity portfolio containing practical labs developed during a Cybersecurity Specialization course, including vulnerability scanning, network forensics, centralized logging, SIEM (Wazuh), threat detection and penetration testing exercises.
---

# Projects

## Vulnerability Scanning Lab
Installation and configuration of vulnerability scanning tools and comparison of their results.

Technologies used:
- Greenbone OpenVAS
- Nessus
- Kali Linux
- Windows Server
- CentOS

---

## Network Forensics Investigation
Investigation of a security incident through PCAP analysis.

Tools:
- Wireshark

Activities:
- identify compromised host
- analyse network traffic
- reconstruct attack chain

---

## Centralized Logging Infrastructure
Deployment of a centralized logging platform.

Technologies:
- Syslog-ng
- NXLog
- Grafana
- Loki

---

## SIEM and Threat Detection Lab
Deployment of a SIEM environment.

Technologies:
- Wazuh
- Suricata
- Nmap
- OWASP ZAP

---

## Capture The Flag / Penetration Testing Lab

Tools:
- Netdiscover
- Nmap
- Nikto

Activities:
- network discovery
- service enumeration
- vulnerability exploitation
- privilege escalation
